# Counter App

# Built with
 - ReactJS
 - Redux
 - Material-UI

# Requirements
 - NodeJS v7+
 - NPM < v5.2 && > v4

# Install
```
$ git clone https://github.com/shierro/counter-app
$ cd counter-app && npm i
```

## Start API First
```
$ npm start:api
```
## Start frontend - Dev mode
```
$ npm run start:dev
```
## Run frontend with production grade code
```
$ npm run build
$ npm start
```

# Run unit test
`
$ npm test
`
